LucidArtists iframe setup

1. Upload waveform-generator.html to the root of the LucidArtists repo.
2. In GitHub repo Settings > Pages:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /(root)
3. Save.
4. Wait for Pages to publish.
5. Your file URL should be:
   https://drewg38.github.io/LucidArtists/waveform-generator.html

Squarespace:
1. Add a Code Block.
2. Paste the contents of squarespace-iframe-embed.html
3. Save and test the published page in an incognito window.
